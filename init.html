init Command

Description:

The init command stands for "initialization" and is used to manage the system's runlevel or reboot the system. It plays a crucial role in controlling the system state and which services are active. System administrators commonly use it to transition the system between different operational modes during startup or shutdown.

Usage:

The init command allows administrators to change the system's runlevel, affecting which services and processes are running. The syntax is as follows:

init [runlevel]

Common runlevels include:

  • 0: System halt (shutdown).
  • 1: Single-user mode (maintenance).
  • 3: Multi-user mode without a graphical interface.
  • 5: Multi-user mode with a graphical interface.
  • 6: Reboot the system.

Example:

init 3

This command switches the system to runlevel 3, enabling multi-user mode without a graphical interface.
