groupmod Command

Description:

The groupmod command is used to modify an existing group on a Linux system. It allows administrators to change a groupâ€™s name, GID (Group ID), or other attributes.

Usage:

The syntax for the groupmod command is:

groupmod [options] group_name

For example:

sudo groupmod -n new_group_name old_group_name

This changes the group name from old_group_name to new_group_name.

Example:

sudo groupmod -g 1001 developers

This command changes the GID of the developers group to 1001.
