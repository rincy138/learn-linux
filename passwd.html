passwd Command

Description:

The passwd command is used to change a user's password in the system. It can be used by the user to change their own password or by the root user to change the password of other users.

Usage:

This command prompts the user to enter a new password. The root user can also use it to reset passwords for other users.

Examples:

Example 1: Change Your Own Password

passwd

This command allows a user to change their own password.

Example 2: Change Another User's Password (Admin)

sudo passwd username

This command changes the password for the specified user username.

Common Use Cases:

  • Changing your password for security purposes.
  • Resetting passwords for users as an administrator.
  • Forcing users to update their passwords regularly to maintain system security.
