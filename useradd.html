useradd Command

Description:

The useradd command is used to create a new user account on a Linux or UNIX-based system. This command allows system administrators to define several user attributes, including the user's home directory, login shell, user ID, and group memberships. It is a vital tool for user management in a multi-user environment, as it helps automate the process of adding new users to the system.

Usage:

The useradd command is used to add new user accounts to the system. It allows the administrator to specify important options, such as the user’s default shell, home directory, and initial groups. A typical syntax is as follows:

useradd [options] username

Common options for the useradd command include:

  • -m: Creates the user's home directory if it doesn't exist.
  • -s: Specifies the user's login shell (e.g., /bin/bash).
  • -G: Adds the user to supplementary groups.
  • -u: Specifies the user’s ID (UID).

Example:

useradd -m -s /bin/bash newuser

This command creates a new user named newuser, with a home directory and a bash shell.
