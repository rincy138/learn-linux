userdel Command

Description:

The userdel command is used to delete a user account from the system. It removes the userâ€™s home directory, mail spool, and other files associated with the account.

Usage:

This command is typically used by system administrators to remove a user account after their access is no longer needed.

Example:

This command removes the user username and their associated files, including the home directory.
